# Glossary

| Term | Definition |
|------|-----------|
| **USV** | Unmanned Surface Vehicle, autonomous boat operating on water surface |
| **DRL** | Deep Reinforcement Learning, RL with deep neural networks as function approximators |
| **DQN** | Deep Q-Network, value-based RL algorithm using neural network to approximate Q-function |
| **DDQN** | Double DQN, uses online network for action selection and target network for evaluation to reduce overestimation |
| **D3QN** | Dueling Double DQN, separates Q-value into state value V(s) and advantage A(s,a) |
| **ID3QN** | Improved D3QN, D3QN + PER + reward shaping + soft target update + LR schedule |
| **PPO** | Proximal Policy Optimization, on-policy policy gradient algorithm |
| **PER** | Prioritized Experience Replay, samples transitions with higher TD error more frequently |
| **MDP** | Markov Decision Process, mathematical framework for sequential decision making |
| **Gymnasium** | Python library providing standard API for RL environments (successor to OpenAI Gym) |
| **SB3** | Stable-Baselines3, popular RL algorithm library compatible with Gymnasium |
| **ML-Agents** | Unity's machine learning toolkit for training agents in Unity environments |
| **Fossen Model** | Norwegian school of ship maneuvering, decomposes forces by physical property (M, C, D, g) |
| **MMG Model** | Japanese school of ship maneuvering, decomposes forces by source (Hull, Propeller, Rudder) |
| **PIRL** | Physics-Informed Reinforcement Learning, incorporating physics knowledge into RL training |
| **PINN** | Physics-Informed Neural Networks, neural networks with PDE residuals as loss constraints |
| **Sim-to-Real** | Transferring policies trained in simulation to real-world deployment |
| **Domain Randomization** | Randomizing simulation parameters during training to improve policy robustness |
| **ROS** | Robot Operating System, middleware for robot software development |
| **Embodied Space** | Unified modeling space: E = (S, A, T, C, P), integrating state, action, task, constraint, and physical environment |
| **GEM** | Geometry-Environment-Mission, three-element structure model for Embodied Space |
| **VLA** | Vision-Language-Action model, foundation model combining visual understanding, language reasoning, and action generation |
| **COLREGS** | Convention on the International Regulations for Preventing Collisions at Sea |
| **Digital Twin** | Virtual replica of physical system, synchronized in real-time |
| **CPS** | Cyber-Physical Systems, integration of computation and physical processes |
