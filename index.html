Project Overview

This project proposes an Improved Dueling Double DQN (ID3QN) algorithm for unmanned surface vehicle (USV) autonomous navigation in the C4 Intelligent Ship Competition scenario. ID3QN integrates prioritized experience replay, composite reward shaping, soft target updates, and learning rate scheduling, achieving 83% success rate in a randomized obstacle environment.

Simulation

20m x 20m water area, 3-5 random obstacles, 180-ray LiDAR, 184-dim state space, 5 discrete actions

Algorithms

4 progressive variants: DQN -> DDQN -> D3QN -> ID3QN, enabling systematic comparison

Training

2000 episodes with GitHub Actions CI/CD, fully reproducible with configurable seeds

Paper

IEEE Transactions format, 5 sections, 8 figures, 14 tables, 20 references

ID3QN Algorithm

Key Improvements over D3QN

Prioritized Replay

SumTree-based PER with O(log N) sampling. High TD-error transitions replayed more frequently. IS weight annealing (beta: 0.4->1.0).

Reward Shaping

3-component dense reward: distance progress + heading alignment + safety penalty. Transforms sparse C4 signals into continuous guidance.

Soft Update

Polyak averaging (tau=0.005) replaces hard copy. Smoother target network transitions reduce value oscillation.

LR Scheduling

StepLR decay: halve every 500 episodes. Fast exploration early, fine-tuning later. Huber loss + gradient clipping.
