    研究方向¶

    围绕 c4nav-core，我们规划了四个阶段的研究路线，从基线论文逐步走向实船验证。

    四个阶段¶

    阶段 名称 做什么 核心改动 产出
    1 基线论文 用现有 c4nav-core + ID3QN 完成论文 无代码改动 论文：ID3QN USV 导航
    2 PIRL 升级 向观测和奖励注入物理信息 (184D->190D) 修改 3 个文件 论文：USV DRL 的 PIRL
    3 仿真到实船 基线策略部署到 SpaceR-USV + 真实 USV130 ROS 封装 论文：Sim-to-Real 对比
    4 PIRL + 实船 用第二阶段 PIRL 版本重复第三阶段测试 合并 2+3 论文：分层迁移

    阶段关系¶

    第一阶段: 基线论文 (当前)
  |
  |  完成论文后，两条路并行推进:
  |
  +---> 第二阶段: PIRL 升级                 +---> 第三阶段: 仿真到实船
  |     在 c4nav-core 中注入物理信息         |     对接 SpaceR-USV + 实船
  |     观测扩展 (B) + 物理奖励 (C)          |     测量 sim-to-real gap
  |     改 3 个文件，训练速度不变             |     需要 ROS + 实船硬件
  |                                          |
  +------------- 汇合 ---------------------->+
                    |
              第四阶段: PIRL + 实船
              PIRL 策略 vs 基线策略
              在实船上的迁移质量对比
              回答: 物理感知能否缩小 gap
    • 第一阶段优先：无需代码改动，可立即完成
    • 第二阶段和第三阶段并行：二改代码，三准备实船，互不阻塞
    • 第四阶段依赖二和三都完成

    逐阶段物理覆盖¶

    每个阶段在 PIRL 四个注入位置 上的覆盖情况：

    阶段 A: 动力学模型 B: 观测空间 C: 奖励函数 D: 网络/损失
    1 基线 一阶惯性 184D 几何量 距离+安全+航向 无
    2 PIRL 一阶惯性（不变） 190D + 物理量 + 能量 + 急转 无
    3 实船 Fossen + 风/浪/流 按 SpaceR-USV 按 SpaceR-USV 无
    4 PIRL+实船 Fossen + 风/浪/流 + 物理量 + 能量 + 急转 无

    四个研究方向¶

    每个阶段对应一个或多个研究方向：

    物理信息强化学习 (PIRL)¶

    对应阶段：第二阶段

    核心问题：物理知识能否在不借助高保真仿真的情况下提升策略？

    在 c4nav-core 的轻量环境中，通过观测扩展（暴露速度/加速度等物理量）和物理感知奖励（能量消耗、急转惩罚）让智能体"理解"物理，训练出更平滑、更接近实船行为的策略。

    分层仿真迁移框架¶

    对应阶段：第三阶段 + 第四阶段

    核心问题：低保真预训练能否加速高保真收敛？

    在 c4nav-core（快，~2000 步/秒）中预训练，迁移到 SpaceR-USV（慢，~50 步/秒）中微调，最终部署到实船。该方法在 USV 领域尚属空白。
